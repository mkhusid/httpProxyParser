# Http Proxy Parser

##Requirements
    $ node version - v12.17.0 (or close to it)
    $ typescript
    $ yarn or npm

## Install

    $ cd PROJECT_TITLE
    $ yarn install
    
## Build before runnig the script

    $ yarn build

## Run the script with specified log file path

    $ yarn start log_example.log

## Run unit tests

    $ yarn test
